## Part 2: 组件规范

### 底部 TabBar

- 4 个 Tab：首页（🏠）、陪诊师预约（👨‍⚕️）、患者管理（📋）、我的（👤）
- 选中态：primary 色图标 + 文字，未选中：#999
- 高度 100rpx，适配 iPhone 底部安全区
- 顶部 1rpx 分割线

### 卡片

- 背景 #FFF，圆角 16rpx，阴影 card
- 内边距 24rpx
- 卡片间距 20rpx

### 按钮

- 主按钮：primary 背景，白字，圆角 full，高度 88rpx
- 次要按钮：白底 + primary 边框 + primary 文字
- 危险按钮：danger 背景
- 按下态：背景色加深 10%
- 禁用态：40% 透明度

### 标签 Tag

- 圆角 full，内边距 4rpx 16rpx，字号 22rpx
- 状态配色：待确认(#FAAD14)、已支付(#1890FF)、进行中(#008000)、已完成(#52C41A)、已取消(#FF4D4F)

### 空状态

- 居中插图（浅色医疗主题插画）
- 主文案 28rpx #666
- 副文案 24rpx #999
- 可选操作按钮

### 加载态

- 列表：骨架屏（灰色色块模拟卡片结构）
- 按钮：Loading 旋转图标 + 文字

## Part 3: 页面设计指令

### P0 — 核心页面（优先设计）

#### 1. 首页

从上到下的布局：

1. **状态栏 + 城市选择**：左侧城市名 + 下拉箭头（primary 色），右侧搜索图标
2. **搜索栏**：圆角搜索框，placeholder "搜索医院、服务"
3. **Banner 轮播**：圆角 16rpx，宽高比 2:1，底部圆点指示器
4. **服务入口**：2 行 × 3 列网格，每个入口：圆形图标（primary-light 底色 + primary 图标）+ 文字标签。6 个入口：陪诊服务、代问诊、病理会诊、代办服务、就医规划、更多
5. **医院列表**：标题"附近医院" + 右侧"查看全部"。医院卡片：左侧医院封面图（圆角 12rpx），右侧名称 + 等级标签（Tag 样式）+ 地址（📍图标 + 灰色文字）+ 特色科室标签

#### 2. 个人中心

两种状态：

**未登录态**：
- 顶部大绿色渐变背景区域（#008000 → #00A86B）
- 中央默认灰色圆形头像 120rpx
- 下方绿色圆角按钮"授权手机号登录"

**已登录态**：
- 顶部绿色渐变背景区域
- 左侧圆形头像 80rpx，右侧昵称（白字 32rpx 加粗）+ 手机号（白字 24rpx 半透明）
- 白色卡片浮出，圆角 24rpx 24rpx 0 0

**菜单列表**（白色卡片内）：
- 普通用户：我的订单（📋）、使用说明（📖）、订阅服务进度（🔔）、退出登录（🚪）
- 陪诊师：我的陪诊（👨‍⚕️）、我的收入（💰）、编辑资料（✏️）、使用说明（📖）、订阅服务进度（🔔）、退出登录（🚪）
- 每行：左侧图标（primary-light 圆形底）+ 标题 + 右侧箭头
- 退出登录行：文字 danger 色

**底部**：
- 普通用户：绿色描边按钮"成为陪诊师，开启灵活就业"
- 右下角浮动客服圆形按钮（绿色，📞图标，阴影 float）

#### 3. 陪诊师预约列表

1. **顶部**：标题"陪诊师预约" + 城市筛选标签（可横滑）
2. **陪诊师卡片**：
   - 白色卡片，内边距 24rpx
   - 左侧：圆形头像 96rpx
   - 右侧上部：姓名（H3 加粗）+ 评分（⭐ 4.8）
   - 右侧中部：服务次数（"服务 128 次"）+ 擅长标签（Tag，最多 3 个，超出显示"+N"）
   - 右侧下部：绿色圆角按钮"立即预约"（小号）
3. **浮动按钮**：右下角"成为陪诊师"，绿色，圆角 full，阴影 float

#### 4. 订单列表

1. **顶部 Tab**：横滑标签栏，选中态：primary 色文字 + 底部 4rpx primary 色指示条，未选中：#666
2. **订单卡片**：
   - 顶部：订单号（caption 灰色）+ 右侧状态 Tag
   - 中部：服务类型（H3）+ 陪诊师信息（头像 + 姓名）
   - 下部：服务时间（🕐图标）+ 金额（primary 色 H2 加粗）
   - 底部操作区：根据状态显示不同按钮（取消/确认收货/评价）
3. **空状态**：医疗主题插画 + "暂无订单" + "去预约陪诊服务"按钮

### P1 — 重要页面

#### 5. 服务详情

1. 顶部服务封面图（全宽，宽高比 2:1）
2. 服务名称（H1）+ 价格（primary 色 H1，带 ¥ 前缀）
3. Tab 切换：服务内容 / 服务流程 / 退款说明
4. 服务内容：富文本段落
5. 服务流程：竖向 Steps 组件，每个节点：圆形序号（primary 背景白字）+ 连接线 + 标题 + 描述
6. 底部固定栏：左侧价格展示 + 右侧"立即预约"主按钮

#### 6. 医院详情

1. 顶部医院封面图（全宽，底部渐变遮罩）
2. 医院名称（H1）+ 等级 Tag（如"三甲"）
3. 信息行：📍 地址（可点击导航，primary 色文字）| 📞 电话（可点击拨打）
4. 特色科室：横滑 Tag 列表
5. 医院介绍：可展开/收起的文本（默认 3 行，展开后全文）
6. 可预约服务列表：卡片式，左侧服务名 + 描述，右侧价格 + 预约按钮

#### 7. 预约下单

1. 已选服务信息卡片（顶部，浅绿底色）
2. 表单区域（白色卡片）：
   - 城市选择（带箭头的 Cell）
   - 医院选择（带箭头的 Cell）
   - 陪诊师选择（带箭头的 Cell）
   - 预约日期时间（日期选择器）
   - 联系人（输入框）
   - 联系电话（输入框）
   - 备注（多行文本框）
3. 服务协议：Checkbox + 链接文字
4. 费用明细卡片：服务费、优惠、合计（primary 色加粗）
5. 底部固定：主按钮"提交订单并支付"

#### 8. 患者管理列表

1. 患者卡片：
   - 白色卡片，左侧彩色竖条（4rpx，primary 色）
   - 姓名（H3 加粗）+ 性别 Tag + 年龄
   - 联系方式（caption 灰色）
   - 右侧箭头
2. 左滑露出红色删除按钮
3. 浮动"+"按钮（右下角，primary 背景白字，阴影 float）
4. 空状态：插画 + "还没有患者信息" + "添加第一位患者"按钮

### P2 — 次要页面

#### 9. 患者详情

1. 顶部患者信息卡片（姓名、性别、年龄、身高、体重）
2. 功能入口网格（2 列）：
   - 📋 病程时间线
   - 📄 生成病历报告
   - ➕ 添加事件
3. 最近事件预览列表（最多 3 条）

#### 10. 病程时间线

1. 顶部筛选栏：日期范围选择 + 事件类型下拉
2. 竖向时间轴：
   - 左侧：时间线（竖线 + 圆点，圆点颜色按事件类型区分）
   - 右侧：事件卡片（日期、类型 Tag、标题、摘要）
   - 事件类型颜色：治疗(#1890FF)、检查(#52C41A)、用药(#FAAD14)、通用(#999)
3. 浮动"+"按钮添加事件

#### 11. 问诊病历报告

1. 仿纸质报告排版：
   - 头部：公司名（左侧）+ 二维码（右侧）+ 报告标题（居中）
   - 患者体格：姓名、年龄、身高、体重，横向排列
   - 各区块：左侧 4rpx primary 色竖条 + 标题（H3）+ 内容
   - 区块包括：疾病诊断、治疗总结、基础病史、医疗事件（按阶段分组）、检查报告
2. 底部固定"下载 PDF"按钮

#### 12. 陪诊师收入管理

1. 顶部概览卡片（绿色渐变背景）：
   - 本月收入（大字白色）
   - 总收入 + 订单数（小字半透明白色）
2. 筛选栏：月份选择器 + 状态筛选（全部/已到账/未到账）
3. 收入明细列表：订单号、金额（primary 色）、状态 Tag、时间

#### 13. 陪诊师注册

1. 顶部步骤指示器（3 步：基本信息 → 身份认证 → 服务信息）
2. 当前步骤表单（白色卡片内）
3. 底部按钮：上一步（次要）+ 下一步/提交（主按钮）

#### 14. 陪诊师资料编辑

1. 头像上传（居中，圆形，带编辑图标覆盖层）
2. 表单：昵称、服务标签（多选 Tag）、服务介绍（多行文本）
3. 底部固定"保存"按钮

## Part 4: 微交互与状态

| 场景 | 处理方式 |
|------|---------|
| 按钮点击 | 缩放 0.96 + 背景色加深，150ms ease |
| 卡片点击 | 缩放 0.98，100ms |
| 页面切换 | 从右向左滑入，300ms |
| 列表加载 | 骨架屏闪烁动画（pulse） |
| 下拉刷新 | 顶部 Loading 旋转图标 |
| 空状态 | 淡入 + 轻微上移动画 |
| Toast 提示 | 从顶部滑入，2s 后自动消失 |
| 弹窗 | 底部滑入 + 背景遮罩淡入 |
