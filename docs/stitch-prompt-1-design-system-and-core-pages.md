# 壹鹿康行 — Stitch UI 设计（第一轮：设计系统 + 核心页面）

## 品牌与调性

- 品牌名：壹鹿康行
- 行业：医疗陪诊服务（患者预约陪诊师陪同就医）
- 调性：专业、温暖、可信赖
- 视觉参考：丁香医生的专业感 + 微医的服务流程感 + 好大夫在线的医患信任感
- 整体风格：现代简洁的医疗健康风格，圆角卡片式布局，柔和的绿色系，适当的阴影和留白

## 设计系统

### 色彩

- 主色：#008000（绿色），用于按钮、TabBar 选中、价格、品牌标识
- 主色浅底：#e8f5e9，用于标签底色、高亮背景
- 主色深色：#006400，用于按钮按下态
- 页面背景：#F5F6FA（淡灰蓝）
- 卡片背景：#FFFFFF
- 标题文字：#1A1A1A
- 正文文字：#666666
- 辅助文字：#999999
- 分割线：#EEEEEE
- 状态色：待确认 #FAAD14（橙）、已支付 #1890FF（蓝）、进行中 #008000（绿）、已完成 #52C41A（青绿）、已取消 #FF4D4F（红）
- 评分星级：#F89D00（金色）

### 字体

- 页面标题：36rpx，700 字重
- 卡片标题 / 价格：32rpx，600 字重
- 列表项标题：28rpx，600 字重
- 正文：28rpx，400 字重
- 辅助信息 / 标签：24rpx，400 字重
- 时间戳 / 状态文字：22rpx，400 字重

### 间距与圆角

- 卡片内边距：24rpx
- 卡片间距：20rpx
- 卡片圆角：16rpx
- 标签圆角：药丸形（全圆角）
- 按钮圆角：药丸形
- 页面左右边距：24rpx

### 阴影

- 卡片：0 4rpx 16rpx rgba(0,0,0,0.06)
- 浮动按钮 / 弹窗：0 8rpx 24rpx rgba(0,0,0,0.12)

### 头像尺寸

- 列表小头像：64rpx 圆形
- 个人中心头像：80rpx 圆形
- 陪诊师详情头像：120rpx 圆形

## 全局组件

### 底部 TabBar

4 个 Tab，选中态绿色图标 + 绿色文字，未选中灰色。高度 100rpx，适配 iPhone 底部安全区，顶部 1rpx 分割线。

- 首页 🏠
- 陪诊师预约 👨‍⚕️
- 患者管理 📋
- 我的 👤

### 按钮

- 主按钮：#008000 背景，白字，药丸形，高度 88rpx
- 次要按钮：白底 + #008000 边框 + #008000 文字
- 危险按钮：#FF4D4F 背景
- 按下态：背景色加深 10%，缩放 0.96，150ms
- 禁用态：40% 透明度
- 微信客服按钮：使用 openType='contact'，样式同普通按钮，图标为 📞

### 标签 Tag

药丸形，内边署 4rpx 16rpx，字号 22rpx。不同状态用不同背景色（见上方状态色）。

### 空状态

居中医疗主题浅色插画 + 主文案 28rpx #666 + 副文案 24rpx #999 + 可选操作按钮。

### 加载态

列表用骨架屏（灰色色块模拟卡片结构，pulse 闪烁动画），按钮用 Loading 旋转图标。

### Dialog 弹窗

- 标题 32rpx 加粗居中
- 内容区 28rpx #666
- 底部两个按钮：左侧"取消"（灰色文字）+ 右侧"确认"（primary 色文字）
- 圆角 24rpx，宽度 80% 屏幕

### Toast 提示

- 居中黑色半透明圆角气泡
- 图标 + 文字
- 2s 自动消失

### ActionSheet 底部操作面板

- 底部弹出，圆角 24rpx 24rpx 0 0
- 选项列表（每行等高，居中文字）
- 底部安全区间距
- 取消按钮（独立一行，上方 8rpx 间距分隔）

## P0 核心页面（本轮设计）

### 页面 1：首页

从上到下布局：

1. **顶部状态栏**：左侧城市名 + 下拉箭头（绿色），右侧搜索图标
2. **搜索栏**：圆角搜索框，placeholder "搜索医院、服务"
3. **Banner 轮播**：圆角 16rpx，宽高比 2:1，底部圆点指示器
4. **引导文案**：左侧"还不清楚什么是陪诊？" + 右侧"点击查看陪诊介绍 >"（绿色可点击文字，跳转服务介绍页）
5. **服务入口**（非 6 图标网格，实际布局如下）：
   - 上排 2 个大按钮（各占一半宽度）：左侧"半天陪诊"（图标 + 标题 + 描述"高效就医 贴心陪伴"）、右侧"代问诊"（图标 + 标题 + 描述"安全省心 高效便捷"），两按钮间有竖向分割线
   - 下排 3 个小按钮（等宽）：全天陪诊、病理会诊、代办服务（每个：小图标 + 文字）
   - 所有按钮点击跳转对应服务详情页
6. **合作医院列表**：标题"附近医院" + 右侧"查看全部"。医院卡片：左侧封面图（圆角 12rpx），右侧名称 + 等级标签（如"三甲"Tag）+ 地址（📍图标 + 灰色文字）+ 特色科室标签

### 页面 2：个人中心

三种状态：

**未登录态**：
- 顶部大绿色渐变背景区域（#008000 → #00A86B）
- 中央默认灰色圆形头像 120rpx
- 下方绿色圆角按钮"授权手机号登录"（微信 getPhoneNumber 授权）

**普通用户已登录态**：
- 顶部绿色渐变背景，左侧圆形头像 80rpx，右侧昵称（白字 32rpx 加粗）+ 手机号（白字 24rpx 半透明）
- 白色卡片浮出（圆角 24rpx 上方圆角），菜单列表：
  - 我的订单 📋 → 跳转订单列表
  - 使用说明 📖 → 下载操作指引文档
  - 订阅服务进度 🔔 → 调用微信订阅消息
  - 退出登录 🚪 → 确认弹窗后退出（文字红色）
- 每行：左侧图标（浅绿圆形底）+ 标题 + 右侧箭头
- 底部"成为陪诊师"浮动卡片组件：绿色背景圆角卡片，左侧 🏥 图标，右侧标题"加入陪诊师家庭" + 副标题"点击了解更多"，点击跳转注册页（需先登录校验）
- 右下角浮动客服圆形按钮（绿色，📞图标，微信 openType='contact'）

**陪诊师已登录态**：
- 同上，但菜单项不同：
  - 我的陪诊 👨‍⚕️ → 跳转陪诊师订单列表（chaperonList，非用户端 orderList）
  - 我的收入 💰 → 跳转收入管理
  - 编辑资料 ✏️ → 跳转陪诊师资料编辑
  - 使用说明 📖
  - 订阅服务进度 🔔
  - 退出登录 🚪
- 无"成为陪诊师"浮动卡片

### 页面 3：陪诊师预约列表

1. **顶部**：标题"陪诊师预约" + 城市筛选标签（可横滑）
2. **陪诊师卡片**：白色卡片，左侧圆形头像 96rpx，右侧姓名（加粗）+ 评分（⭐ 4.8）+ 服务次数（"服务 128 次"）+ 擅长标签（Tag，最多 3 个，超出显示"+N"）+ 右下角绿色小按钮"立即预约"
3. **右下角浮动按钮**"成为陪诊师"，绿色药丸形

### 页面 4：订单列表

1. **顶部 Tab 栏**：横滑标签（全部/待确认/已支付/已接单/进行中/已取消/已完成），选中态绿色文字 + 底部 4rpx 绿色指示条，未选中灰色
2. **订单卡片**：
   - 顶部：订单号（灰色小字）+ 右侧状态 Tag
   - 中部：服务类型（标题）+ 陪诊师信息（小头像 + 姓名）
   - 下部：服务时间（🕐图标）+ 金额（绿色大字加粗）
   - 底部操作按钮区（根据状态动态显示）：
     - 待确认/已支付：显示"取消订单"按钮
     - 已支付/已接单：显示"取消订单"按钮（点击弹出退款规则 Dialog：>24h 退 100%、12-24h 退 90%、4-12h 退 50%、<4h 退 20%、已开始不退款，展示退款金额 + 确认/取消）
     - 进行中：显示"确认收货"按钮
     - 已完成：显示"评价服务"按钮（跳转评论页）
     - 已取消：无操作按钮
3. **空状态**：医疗主题插画 + "暂无订单" + "去预约陪诊服务"按钮

### 页面 5：评论页（订单评价）

1. **订单信息卡片**（顶部白色卡片）：服务名称、陪诊师姓名、服务时间、服务费用
2. **评分区域**：
   - 标题"服务评分"
   - 评分描述文字（根据星级动态显示：未选→"请为本次服务打分"，1 星→"非常差"，2 星→"差"，3 星→"一般"，4 星→"好"，5 星→"非常好"，描述文字金色加粗）
   - 5 颗星评分组件（⭐，金色选中 #F89D00，灰色未选 #E0E0E0）
3. **评论内容区域**：
   - 标题"评论内容"
   - 多行文本框（6 行高，placeholder "请输入您的评论内容（可选，最多500字）"）
   - 底部字数统计（"已输入 / 500"）
4. **提交按钮**：主按钮"提交评论"，未选评分时禁用（40% 透明度）

### 页面 6：服务介绍页（营销落地页）

这是一个纯展示型的营销页面，从上到下：

1. **顶部 Banner 图**（全宽封面图）
2. **"壹鹿康行能解决哪些问题？"** 标题区域
3. **"陪诊是什么？"** 文案段落：解释陪诊服务的定义和价值
4. **问题场景网格**（2 列 × 6 行，每个格子：标题 + 描述）：
   - 就医流程复杂 → 陪诊师理清流程，确保顺利就诊
   - 排队等待 → 陪诊师代排队，减少等待时间
   - 缺乏医疗知识 → 提供专业指导，协助选医和理解治疗方案
   - 沟通障碍 → 协助医患沟通，避免误解
   - 医院环境复杂 → 导航指引，避免迷路
   - 病历整理 → 整理病历，跟进病情
   - 缺乏陪伴 → 提供情感支持，缓解焦虑
   - 异地就医 → 提供本地导航和支持
   - 特殊人群 → 帮助老年人及行动不便者顺利就医
   - 减少医患纠纷 → 促进医患理解，降低纠纷风险
   - 跨国就医 → 解决语言文化障碍，确保顺畅沟通
   - 流程陪伴 → 陪同独自就诊者，全程引导就医流程
5. **"为什么选壹鹿康行？"** 标题区域
6. **品牌优势描述**："壹鹿康行专注陪诊服务，从客户的角度出发，不断改进完善服务细节，力求提供最完美的服务。"
7. **优势列表**（每项：图标 + 标题 + 描述，纵向排列）：
   - 独创全方位陪诊：为每位就诊人配备一名陪诊师，提供全方位的专业服务
   - 精准分诊：根据患者的症状和需求，精准推荐合适的科室和医生
   - 病情跟进：就医后陪诊师定期跟进康复情况
   - 专业陪诊师：具备医学知识和实践经验，提供基本健康指导
   - 系统性健康档案：为每位患者建立详细的健康档案
   - 标准专业化服务流程：完善且标准化的陪诊服务流程
   - 完整且丰富的陪诊环节：诊前、诊中、诊后的全方位健康管理
8. **有/无陪诊对比**（两个对比卡片）：
   - 左侧"无陪诊"（❌ 红色图标）：就诊人因流程不熟悉浪费大量时间，感到焦虑和无助
   - 右侧"有陪诊"（✅ 绿色图标）：陪诊师引导下流程顺畅高效，平均节省一半就诊时间
9. **三步预约引导**：
   - 步骤 1：选择所需的服务
   - 步骤 2：填写联系方式和姓名，支付预约费用
   - 步骤 3：保持电话通畅，陪诊助理将在第一时间联系您
10. **底部按钮**："返回选择服务"（绿色按钮，返回上一页）

## 微交互

- 按钮点击：缩放 0.96 + 背景色加深，150ms ease
- 卡片点击：缩放 0.98，100ms
- 页面切换：从右向左滑入，300ms
- 列表加载：骨架屏 pulse 闪烁
- 下拉刷新：顶部 Loading 旋转图标
- 空状态：淡入 + 轻微上移
- Toast 提示：从顶部滑入，2s 自动消失
- 弹窗（Dialog）：居中缩放弹入 + 背景遮罩淡入
- ActionSheet：底部滑入 + 背景遮罩淡入
- 评分星级：点击时缩放 1.2 弹跳动画，150ms
