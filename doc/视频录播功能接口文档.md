# 医疗陪护小程序 - 视频录播功能接口文档

## 1. 接口概述

本接口文档定义了医疗陪护小程序视频录播功能所需的所有API接口，包括课程管理、视频播放、权限控制、我的课程等功能模块。

## 2. 接口基础信息

### 2.1 基础URL
- 开发环境：`https://caxcz.xin`
- 测试环境：`https://caxcz.xin`
- 生产环境：`https://caxcz.xin`

### 2.2 请求头
所有请求需携带以下请求头：
```json
{
  "Content-Type": "application/json",
  "Authorization": "{token}"  // 用户认证令牌
}
```

### 2.3 响应格式
所有响应采用统一格式：
```json
{
  "code": 200,              // 状态码，200表示成功，非200表示失败
  "msg": "success",   // 状态信息
  "data": {}              // 响应数据
}
```

## 获取课程列表

### 基本信息

**接口地址**: `/api/courses`  
**请求方式**: `GET`  
**接口描述**: 获取课程列表，支持分页、分类、搜索  

### 请求参数

| 参数名称 | 参数说明 | 请求类型 | 是否必须 | 数据类型 | 示例值 |
|---------|---------|---------|---------|---------|-------|
| page | 页码，默认1 | query | 否 | Integer | 1 |
| pageSize | 每页数量，默认10 | query | 否 | Integer | 10 |
| categoryId | 分类ID | query | 否 | String | 1 |
| keyword | 搜索关键词 | query | 否 | String | 糖尿病 |
| sortBy | 排序字段，默认createdAt | query | 否 | String | createdAt |

## 获取视频播放地址

### 基本信息

**接口地址**: `/api/courses/{courseId}/videos/{id}/play-url`  
**请求方式**: `GET`  
**接口描述**: 获取指定视频的播放地址，包含签名和过期时间  

### 请求参数

| 参数名称 | 参数说明 | 请求类型 | 是否必须 | 数据类型 | 示例值 |
|---------|---------|---------|---------|---------|-------|
| id | 视频ID | path | 是 | String | 1 |
| courseId | 课程ID | path | 是 | String | 1 |

### 响应结果

``json
{
  "code": 200,
  "msg": "success",
  "data": {
    "videoId": "1",
    "playUrl": "https://example.com/play.mp4?sign=abc123&expire=1234567890",
    "expireTime": 1234567890,
    "quality": "hd"
  }
}
```

### 返回字段说明

| 字段名称 | 字段说明 | 数据类型 |
|---------|---------|---------|
| videoId | 视频ID | String |
| playUrl | 视频播放地址（包含签名和过期时间） | String |
| expireTime | 地址过期时间戳 | Long |
| quality | 视频质量 | String |
