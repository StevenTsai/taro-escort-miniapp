# 医疗陪护小程序 - 视频录播功能需求文档

## 1. 功能概述

### 1.1 功能背景
为了丰富医疗陪护小程序的服务内容，提升用户体验，现计划增加视频录播功能模块，支持播放医疗相关课程，包括课程列表展示、课程详情查看、视频播放、权限控制等功能。

### 1.2 功能目标
- 提供医疗相关课程的视频播放功能
- 实现课程分类和搜索功能
- 支持课程详情查看和视频播放
- 实现权限控制，区分免费和付费课程
- 支持课程收藏和观看历史记录

## 2. 技术栈

## 6. 权限控制

### 6.1 权限类型
- **免费课程**：所有用户均可观看
- **付费课程**：仅购买过该课程的用户可观看
- **会员课程**：仅会员用户可观看

### 6.2 权限检查流程
1. 用户进入视频播放页
2. 检查用户是否登录
3. 检查课程类型
4. 如果是免费课程，直接播放
5. 如果是付费课程，检查用户是否购买
6. 如果是会员课程，检查用户是否为会员
7. 如果权限不足，提示用户购买或开通会员

### 6.3 权限检查实现
```javascript
// 权限检查函数
const checkCoursePermission = async (courseId) => {
  // 检查用户是否登录
  const isLoggedIn = useAuthStore.getState().isLoggedIn;
  if (!isLoggedIn) {
    Taro.navigateTo({ url: '/pages/profile/profile' });
    return false;
  }
  
  // 获取课程信息
  const course = await getCourseDetail(courseId);
  
  // 检查课程类型
  if (course.type === 'free') {
    return true;
  }
  
  if (course.type === 'paid') {
    // 检查用户是否购买
    const hasPurchased = await checkPurchaseStatus(courseId);
    if (!hasPurchased) {
      Taro.showModal({
        title: '提示',
        content: '该课程为付费课程，请先购买',
        success: (res) => {
          if (res.confirm) {
            // 跳转到课程详情页购买
            Taro.navigateTo({ url: `/pages/course/detail?id=${courseId}` });
          }
        }
      });
      return false;
    }
    return true;
  }
  
  if (course.type === 'member') {
    // 检查用户是否为会员
    const isMember = useAuthStore.getState().userInfo?.isMember;
    if (!isMember) {
      Taro.showModal({
        title: '提示',
        content: '该课程为会员专属课程，请先开通会员',
        success: (res) => {
          if (res.confirm) {
            // 跳转到会员开通页
            Taro.navigateTo({ url: '/pages/member/index' });
          }
        }
      });
      return false;
    }
    return true;
  }
  
  return false;
};
```

## 10. 风险评估

### 10.1 技术风险
- **视频播放兼容性**：不同设备、不同微信版本对视频播放的支持可能存在差异
- **视频加载速度**：视频文件较大，可能影响加载速度
- **权限控制安全性**：需要确保权限控制的安全性，防止未授权访问

### 10.2 解决方案
- 采用微信小程序原生 `video` 组件，确保兼容性
- 对视频进行压缩和优化，提高加载速度
- 实现服务端和客户端双重权限检查，确保安全性
