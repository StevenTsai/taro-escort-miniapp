# 壹鹿康行小程序 - 功能列表说明书

## 项目概述

**项目名称：** 壹鹿康行小程序  
**技术栈：** Taro 4.x + React + NutUI + 微信小程序  
**项目类型：** 医疗陪诊服务平台  

## 功能模块总览

### 1. 首页模块

#### 1.1 轮播图展示
- **功能描述：** 展示平台活动、服务推广等轮播广告
- **数据来源：** `/api/marquee` 接口
- **特性：** 自动轮播、手动切换、点击跳转

#### 1.2 服务组件展示
- **功能描述：** 展示平台提供的核心服务项目
- **服务类型：** 陪诊服务、医疗咨询、健康管理等
- **交互：** 点击服务卡片跳转至服务详情页

#### 1.3 医院列表
- **功能描述：** 展示合作医院信息
- **信息包含：** 医院名称、地址、等级、特色科室等
- **交互：** 点击医院卡片查看医院详情

### 2. 陪诊师预约模块

#### 2.1 陪诊师列表
- **功能描述：** 展示可预约的陪诊师信息
- **筛选条件：** 按城市筛选
- **信息展示：** 陪诊师头像、姓名、评分、服务次数、擅长领域
- **交互：** 点击查看陪诊师详情、发起预约

#### 2.2 陪诊师详情
- **功能描述：** 展示陪诊师详细信息
- **信息包含：** 个人简介、服务范围、用户评价、服务价格
- **操作：** 立即预约、收藏陪诊师

#### 2.3 陪诊师注册
- **功能描述：** 陪诊师入驻申请
- **注册流程：** 
  - 第一步：基本信息（姓名、性别、城市、手机号）
  - 第二步：身份认证（身份证正反面上传）
  - 第三步：专业信息（服务标签、服务介绍）
- **功能特性：** 
  - 图片上传与预览
  - 城市选择器
  - 标签多选
  - 分步表单验证

#### 2.4 陪诊师编辑
- **功能描述：** 陪诊师个人信息修改
- **可编辑内容：** 头像、服务标签、服务介绍等

#### 2.5 陪诊师收入
- **功能描述：** 查看陪诊师收入统计
- **数据展示：** 总收入、本月收入、订单数量

### 3. 患者管理模块

#### 3.1 患者列表
- **功能描述：** 管理用户关联的患者信息
- **功能特性：**
  - 分页加载（每页10条）
  - 下拉刷新
  - 上拉加载更多
  - 删除患者
- **权限控制：** 需要登录才能使用

#### 3.2 患者详情
- **功能描述：** 查看患者详细信息
- **信息展示：** 患者基本信息、关联订单、服务记录

#### 3.3 创建患者
- **功能描述：** 添加新的患者信息
- **信息字段：** 姓名、性别、年龄、联系方式、备注等

#### 3.4 患者时间线
- **功能描述：** 记录患者就医过程中的重要事件
- **事件类型：**
  - 治疗小结
  - 检查事件
  - 用药事件
  - 一般事件
- **功能特性：**
  - 时间轴展示
  - 事件筛选（按日期、类型）
  - 分页加载
  - 下拉刷新

#### 3.5 事件详情
- **功能描述：** 查看单个事件的详细信息
- **信息展示：** 事件类型、时间、描述、相关附件

#### 3.6 添加治疗小结
- **功能描述：** 记录患者的治疗情况
- **信息字段：** 治疗日期、医院、科室、医生、治疗内容、效果评估

#### 3.7 添加用药事件
- **功能描述：** 记录患者的用药情况
- **信息字段：** 药品名称、剂量、频次、开始时间、结束时间、注意事项

#### 3.8 添加检查事件
- **功能描述：** 记录患者的检查情况
- **信息字段：** 检查日期、医院、检查项目、检查结果、医生建议

#### 3.9 添加一般事件
- **功能描述：** 记录患者的其他重要事件
- **信息字段：** 事件日期、事件类型、事件描述、相关附件

#### 3.10 报告生成
- **功能描述：** 基于患者时间线生成医疗报告
- **功能特性：**
  - 自动汇总患者就医记录
  - 生成结构化报告
  - PDF导出下载
- **API接口：** `/api/timeline/report/generate/{patientId}`

### 5. AI功能模块

#### 5.1 AI助手
- **功能描述：** 智能对话助手，提供医疗咨询和问答服务
- **技术方案：** 基于混元大模型（hunyuan-exp）
- **功能特性：**
  - 多轮对话
  - 文字输入
  - 语音输入
  - 图片上传
  - 文件上传
  - 网络搜索
  - 思考过程展示
  - 工具调用详情
  - 对话历史记录
  - 下拉刷新
  - 复制功能
- **模型配置：**
  - 模型提供商：hunyuan-exp
  - 快速响应模型：hunyuan-t1-latest
  - 机器人ID：ibot-answer-orsnlb

#### 5.2 AI生图
- **功能描述：** 基于文本描述生成图片
- **功能特性：**
  - 文本提示词输入
  - AI优化提示词
  - 图片生成
  - 图片预览
  - 图片下载保存
  - 频率限制（每分钟1次）
- **API接口：** `/api/image/generate`

#### 5.3 AI图像解析
- **功能描述：** 使用AI解析图片内容
- **技术方案：** 基于微信云开发AI能力
- **功能特性：**
  - 图片上传
  - AI内容识别
  - 解析结果展示
  - 支持环境共享模式
- **SDK版本要求：** >= 3.8.1
